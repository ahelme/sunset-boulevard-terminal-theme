## About

Sunset Boulevard is a dark terminal theme crafted to capture golden hour in the city - where the warm, fading light of sunset meets the electric glow of neon signs. Low-strain visuals for long coding sessions with excellent readability.

## Installation

### For iTerm2 (macOS)
1. Download the `Sunset-Boulevard.itermcolors` file
2. Open iTerm2 ‚Üí Preferences ‚Üí Profiles ‚Üí Colors
3. Click "Color Presets..." ‚Üí Import
4. Select the downloaded `.itermcolors` file
5. Apply the "Sunset Boulevard" preset

### For Terminal.app (macOS)
1. Download the `Sunset-Boulevard.terminal` file
2. Double-click to import into Terminal.app
3. Set as default in Terminal ‚Üí Preferences ‚Üí Profiles

### For Warp (cross-platform)
1. Download files from warp-themes/
2. Follow: https://docs.warp.dev/terminal/appearance/custom-themes 

### For Other Terminals
Check the `/themes` directory for your specific terminal emulator configuration files.
